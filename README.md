# Лабораторын ажил: Git Clone, Commit, Push, Pull ашиглах

## Түлхүүр үг
Git, Github, repository, done, commit, push, pull, remote, repository, local, repository, version control

## ХИяээлийн зорилго
Оюутнуудад Github дээр байрлах repository-г, локаль компюьтер дээр өөрчлөлт хийх, commit хийх, Github руу push хийх болон Github дээрх шинэ өөрчлөлтийг pull хийх процессыг ойлгуулах

## Суралцахуйн үр дүн

Энэхүү лабораторын ажлыг хийхсний дараа оюутан дараах чадваруудыг эзэмшинэ

     1.Github repository-г локаль компьютер clone хийх
     2.Локаль repository дээр файл өөрчлөх
     3.git add болоэ git commit ашиглах
     4.Өөрчлөлтүүдийг Github руу git push ашиглан илгээх
     5.Github дээрх шинэ өөрчлөлтийг git pull ашиглан татах
